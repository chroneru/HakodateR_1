Twitter Account¶

まずは、Twitterアカウントを取得しよう。¶

アカウント作成をクリック！¶

名前と電話番号か、メールアドレスを登録！¶

認証コードを入力しよう!¶

パスワードを入力しよう！¶

Twitter developer にアクセスしよう¶

右上の Sign In をクリック！¶

Twitterにログインしよう！¶

右上の Apply をクリック！¶

Continue をクリック！¶

チェックを入れよう！¶

今回は、個人で使うので下の方にチェックを入れよう！

メールが送られたことを確認しよう！¶

アプリ名を入力しよう！¶

アプリの説明を書こう！¶

読んだら Create ！¶

アプリの詳細を確認してみよう！¶

Access token の方を Create ！¶
